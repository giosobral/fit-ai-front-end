# 🏋️📈📊 Fit.AI

O **Fit.AI** é uma plataforma moderna de acompanhamento fitness que utiliza **inteligência artificial** para auxiliar usuários na gestão de treinos, saúde e evolução física.
A aplicação permite que o usuário registre dados importantes, visualize seu progresso e receba **insights inteligentes gerados por IA**, ajudando na tomada de decisões para melhorar desempenho, saúde e resultados.

Com uma arquitetura moderna e escalável, o sistema foi projetado para oferecer **experiência rápida, segura e intuitiva**, utilizando tecnologias atuais do ecossistema web.

---

# 🚀 Funcionalidades

🤖 **Insights com Inteligência Artificial** – Criação de plano de treinos, e sugestões inteligentes baseadas nos dados do usuário.

🏋️ **Gestão de Treinos** – Registro e acompanhamento das atividades físicas.

📊 **Dashboard de Progresso** – Visualização de métricas importantes relacionadas à evolução do usuário.

🔐 **Autenticação Segura** – Login social com Google utilizando autenticação moderna e segura.

🌐 **Arquitetura Moderna** – Aplicação separada em frontend e API para melhor escalabilidade.
